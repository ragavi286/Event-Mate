# Event-Mate

Event-Mate is a smart event management platform designed to simplify the process of discovering, planning, and managing events. It provides separate interfaces for Clients and Vendors, offering an efficient and organized event experience.

---

## Features

- Client & Vendor role-based login
- Vendor listing and service display
- Event booking and management
- AI-based event suggestions for clients
- Calendar view for scheduled events
- Vendor booking management

---

## Tech Stack

| Technology | Purpose |
|------------|----------|
| HTML / CSS / JavaScript | Frontend design & interactivity |
| Java | Core application logic |
| MySQL | Database for storing event data |

- Web browser (Chrome recommended)
